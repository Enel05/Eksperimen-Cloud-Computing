# Website Rangkuman Introduction to FinOps

Versi ini menggunakan tema **hitam emas** dengan tampilan elegan dan modern.

## Fitur desain
- Tema warna hitam, emas, dan ivory
- Background glow emas
- Efek reveal saat scroll
- Floating cards
- Hover effect pada card dan tombol
- Layout responsif untuk desktop dan mobile

## Struktur file
- `index.html`
- `style.css`
- `script.js`

## Menjalankan lokal
Buka `index.html` langsung di browser, atau gunakan Live Server di VS Code.

## Deploy ke Render
1. Upload folder project ke GitHub.
2. Login ke Render.
3. Pilih **New +** > **Static Site**.
4. Hubungkan repository GitHub.
5. Isi:
   - **Build Command**: kosongkan
   - **Publish Directory**: `.`
6. Klik **Create Static Site**.
